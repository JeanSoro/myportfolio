HTML Email Template

View Project View Code

A fully responsive and modern HTML5 Email template created with no frameworks for a watch company. Using html5 and inline CSS3 in this project allowed me to make sure that styles would be rendered properly on all devices.

The images were optimized using Tinypng, a tool to compress images and decrease their sizes to increase loading speed. Finally, I used Litmus in order to make sure that the email rendered correctly on all email clients ranging from aol to gmail. Using tables to create the display without a framework was the most challenging part of this project.
