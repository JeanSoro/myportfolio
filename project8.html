React Car Dealership

View Project View Code

A fully responsive and modern web app created using a template for luxury cars dealership companies wishing to advertise their products. Using React.js in this project allowed me to display sorted and filtered dummy data that I created to mimic data coming from a back-end server based on user feedback. I also made use of styled components and an higher order component.
