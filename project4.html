Prime Beef Restaurant

View Project View Code

A fully responsive and modern web app created from scratch using a design for a steak house looking to modernize their online pressence. Using Hyperapp.js in this project allowed me to solidify my knowledge of React.js, display dummy data that I created to mimic data coming from a back-end server to populate the app state, thus hydrating the header section, the main menu section, the reviews section buttons and quotes as well as the contact information.

This allows the user to interact with the UI in order to go through the reviews. Learning a light weight, but unpopular Front-End library like Hyperapp.js was a way to confirm to myself that I could pick up new tech fast by referencing the official documentation. Like all my projects, this is ongoing and I will be adding more features, a modal component showing a full menu on user click perhaps.
