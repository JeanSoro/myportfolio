E-Commerce Prototype

View Project View Code

This project started off as a design exercise using html5 and css3 only but ended up using Vanilla Javascript to create a cool E-commerce prototype website. The idea behind the project was to create a modern ui, for a clothing store, where customers could not only view products, but also add, update and remove them from their shopping cart!

the most challenging parts of the project were: The initial set up of a fake rest api using json, using javascript async/await method to then retrieve the data after familiarizing myself with the contentful headless cms/api, creating various javascript classes to interact with different parts of the UI, as well as persisting data through local storage.
