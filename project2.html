SoftUI Product Landing Page

View Project View Code

A fully responsive and modern HTML5 and BootStrap4 landing page using one of the major trends in UX design for 2020: Neumorphism. Neumorphism, or soft UI, is a visual style that combines background colors, shapes, gradients, highlights, and shadows to ensure graphic intense buttons and switches. All that allows achieving a soft, extruded plastic look, and almost 3D styling.

Make sure to hover over the buttons, as well as display boxes on the page to see this UI effect come to life.
