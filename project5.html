React.js E-commerce

View Project View Code

Allow a few seconds for products to load up as initial request to Heroku servers is known for being slow. A cool E-commerce prototype website with core functionalities like user authentication, data persistance using local storage and Stripe payment implementation. The idea behind the project was to create a modern ui, for a clothing store, where customers could not only view products, but also login and out of their shopping session, add, update, remove them from their shopping cart and checkout securely!

the most challenging parts of the project were: interacting with Strapi headless CMS to create and consume rest api routes asynchronously for the products(creating and deleting), as well user authentication features. Also rewriting the application to implement React Hooks API and make decisions as to where to store application state was an enriching experiment.P.S - Hover on the icons below for a cool effect!
