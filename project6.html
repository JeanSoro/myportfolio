Craigslist Clone

View Project View Code

p.s: initial loading might be slow on your machine. A prototype for a modern web app created from scratch while drawing inspiration from the real Craigslist page design. Using React.js and Node.js in this project was my first attempt at building a full stack JS app. As I aspire to become a full stack developer, I am learning a great deal from this ongoing project. I set up a server using the Express.js framework, then created various routes based on requests to api files that I also created to mimic data coming from a back-end server based on user feedback.

Currently, only the for-sale section for cars and trucks is active. The listings are filterable by prices, make and models. Setting routes and manipulating the data coming from the back-end to hydrate the state, thus populating forms fields, as well as displays are the most challenging part of this project. this is an ongoing project and I will be making all sections fully functional as my knowledge of NODE.js increases. It is currently on hold.
