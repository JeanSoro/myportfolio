Toronto Raptors Championship Tour

View Project View Code

A prototype for a modern web app created from scratch based on how I would have wanted the Toronto Raptors to celebrate their 2019 NBA Championship triumph. This project was truly fun to build and used only HTML5 and CSS3. The focus was on making a fully responsive page. As you can see by looking at the source code, Flexbox and CSS Grid were used . As well, This was an opportunity to get comfortable with creating cool background color effects and CSS filters to achieve grainy / throwback photo effects as illustrated in the image of the team holding the trophy. One additional page, as well as "back to top" button might be created to enhance this project in the future.
